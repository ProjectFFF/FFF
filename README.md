# FFF(Find Fine Fit)

> 온라인 쇼핑에서의 **사이즈 미스** 문제

> **유사 체형**을 가진 유저를 찾아

> 서로의 **옷장**을 공유하여

> 사이즈 비교를 가능하게 한다

&nbsp;
